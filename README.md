# SlimeCore Minimal Dev. Utility | `scdev`

**ID:** `sixslime.scdev`

> Small utility for developing with SlimeCore.

## Description

### Listening To Hooks

Players with the `scdev.listen` entity-tag recieve helpful info in chat related to SlimeCore events when they happen (load/entrypoint order on reload, rebuild status messages, etc.).

### User Functions

Offers a command-line friendly wrapper around `slimecore:rebuild`, as well as a function to quickly generate `manifest.pack.dependencies` manifest-function input.
